# vim 基本使用快捷键

## 移动光标
$ 行尾 ^ 行首
w 下一个单词词尾 e 下个单词词首
b 前一个单词
X 退格
ctrl + r 重做
ｓ覆盖
gg H M L G 首行　中间　末尾

- + 上一行下一行
{ } 前一个后一个空行
( ) 当前句子的行首行尾

## 查找
f 查找 F 反向查找
df(

## 搜索
/ ? 前后查找
n N# 下一个上一个
* 直接匹配到当前光标的字符串

## 高级
<< >> 缩进
~ 切换大小写 title r 替换一个字符
ZZ ZQ
zt zb
ctrl + F/B 翻页


# 命令

:split :vsplit

ctrl w HJKL
ctrl w r
ctrl -+<>
ctrl C
